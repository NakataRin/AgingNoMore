# AgingNoMore
## Setup
### THIS IS ONLY NEEDED FOR LOCAL BUILDING OF THE SOURCE CODE. If you simply want to use this mod, go to the release page (or nexus), don't be silly.

Create a folder named `GameRefs` and copy the following files from the game:
* `Assembly-CSharp.dll`
* `UnityEngine.CoreModule.dll`
* `BepInEx.dll`
* `0Harmony.dll`

Project references expect these files to exist locally.
